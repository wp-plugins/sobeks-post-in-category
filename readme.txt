Creates a list of the posts of one or more categories.

== Description ==

List the posts of one or more selected categories. Whether you want more control over your sitemap or add the contents of a category to a post, this plugin can be used to list the contents of one or several categories in alphabetical order. Please read http://wordpress.sobek.pl/sobeks-post-in-category-plugin/ for installation and usage.

== Installation ==

Please visit
- http://wordpress.sobek.pl/sobeks-post-in-category-plugin/ for installation and usage
- http://wordpress.sobek.pl/handmade-wordpress-sitemap-a-tutorial/ for a tutorial on how to create a sitemap
