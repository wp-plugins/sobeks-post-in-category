Creates a list of the posts of one or more categories.

== Description ==

List the posts of one or more selected categories. Get more control over your sitemap or add the contents of categories to a post - this plugin displays a sorted list the posts of one or more categories. Please read 
[read this post](http://wordpress.sobek.pl/sobeks-post-in-category-plugin/ "Sobek's Post in Category")
 for installation and usage.

Can do statements:
* Can display the contents of one category
* Can display the contents of two or more categories as one list
* Can display the contents of two or more categories as seperate lists
* Can display the output as list items i.e. enclosed by "li" tags
* Can sort the output by post title or date
* Can display the number of posts that were listed in one call
* Can be called multiple times in a single page or post



== Version history ==
