Displays a list of posts from one or more categories.

== Description ==

Get more control over your sitemap or add the contents of categories to a post - this plugin displays a sorted list the posts of  categories. Please read [read this post](http://wordpress.sobek.pl/sobeks-post-in-category-plugin/ "Sobek's Post in Category")
 for installation and usage.

**"Can do" statements:**
* Can display the contents of one or more categories as joint or seperate lists 
* Can sort the output by post title, date or number of comments
* Can display the number of posts that were listed in one call and the number of comments for each post 
* Can be called multiple times in a single page or post 


* V1.5 - Improved speed + 2 bugfixes + support for sorting posts by number of comments eg. sobeks_posts_by_category(**commentsort**,12,3,56);
* v1.4 - lets you output the number of comments after the link to a post, also the order of arguments supplied doesn't matter now eg. sobeks_posts_by_category(**comments**,12,3,56);
* v1.3 - now with the option to display how many posts you have listed with one call sobeks_posts_by_category(**count**,12,3,56);
* v1.2 - adds a sort by title or date option and a name change of the function eg. sobeks_posts_by_category(**date**,12,3,56);
* v1.1 - features listing several categories with one call eg. s_posts_by_category(**12,3,56**);
* v1.0 - basic working version eg. **s_posts_by_category(12)**;
